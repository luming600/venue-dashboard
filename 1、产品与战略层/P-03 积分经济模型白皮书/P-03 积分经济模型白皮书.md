## 一、引言：积分不是成本，是"游戏化储值卡"

在传统的商业认知中，积分常被视为一种营销成本。但在杆杆响天梯赛的生态里，积分被重新定义——它是一张**"游戏化储值卡"**，是连接用户行为、球房收益与平台生态的核心纽带。

- **对用户：** 积分是竞技付出的回报，是可兑换、可提现、可炫耀的资产。
- **对球房：** 积分是引导用户行为（延长停留、提升频次、消耗高毛利商品）的精准杠杆。
- **对平台：** 积分是构建跨店流通、数据沉淀、产业连接的信用基础。

本白皮书系统阐述杆杆响积分经济模型的设计逻辑、运行规则与商业价值。

## 二、积分获取规则

积分获取遵循三大核心原则：**时间长、要赢、比赛激烈**。所有规则均围绕这三点展开，确保用户行为与球房利益高度一致。

### 2.1 单场比赛积分计算公式

**胜方获得积分 = 基础局数分 + 额外奖励分 × 球房积分产出比例**

| 项目 | 对应积分 | 说明 |
| --- | --- | --- |
| 每一小局 | 5分 | 按比赛总局数计算，仅胜方获得 |
| 炸清/接清 | +5分 | 奖励一杆清台的精彩技艺 |
| 双赛点 | +10分 | 奖励进入决胜局的焦灼对局 |

**示例：** 一场5:4的比赛（共9局），胜方有一次炸清且进入双赛点，则基础总分 = 9×5 + 5 + 10 = 60分。

### 2.2 球房积分产出比例

球房可根据自身台费标准和运营策略，设置积分产出比例（建议范围0.6倍 - 1.2倍）：

- 台费较高/控制成本：设0.6倍，60分×0.6=36分
- 推广期/激励参与：设1.2倍，60分×1.2=72分

### 2.3 其他获取方式

| 行为 | 奖励积分 | 目的 |
| --- | --- | --- |
| 分享战绩海报 | 30分/次 | 鼓励传播，免费曝光 |
| 邀请好友参赛 | 50分/人 | 低成本拉新 |
| 举报违规 | 50分/次 | 维护公平竞技 |

### 2.4 系统自动发放

比赛对局产生的积分由天梯系统自动计算并发放至胜方账户，无需人工干预，确保高效准确。

## 三、积分消耗与退出通道

必须引导积分消耗在**"高毛利、增营收、锁未来"**的场景，避免替代正常现金销售。

### 3.1 核心消耗场景

| 消耗渠道 | 具体形式 | 设计价值 |
| --- | --- | --- |
| A. 兑换专属特权/服务 | • 闲时段台时券 • 教练体验课 • 器材保养服务 | 盘活闲置资源，不挤占高峰现金收入 |
| B. 兑换高毛利附加品 | • 特调饮品（成本低） • 品牌周边（巧粉、手套） | 提升毛利，强化品牌 |
| C. 游戏化抽奖 | • 积分抽奖池（大奖为球杆、手机等） | 沉淀积分，创造惊喜 |
| D. 虚拟荣耀 | • 虚拟勋章、主页装扮 | 零成本消耗，满足荣誉感 |
| E. 少量提现 | • 设置门槛（如累计胜场≥10场） • 每月限提1次，最低100分起 | 制造终极吸引力，控制现金流出 |

### 3.2 禁止/限制场景

尽量避免积分兑换矿泉水、普通可乐等低毛利、易替代现金销售的商品。球房可根据实际情况设置"不可兑换商品清单"。

## 四、球房财务模型

### 4.1 单球房投入产出测算（以50名新用户为例）

新用户完成10场比赛即可赠送1000积分。

| 项目 | 金额 | 说明 |
| --- | --- | --- |
| 球房投入（积分采购） | 5000元 | 按每用户100元价值采购5000分（1元=10分） |
| 用户为赚积分产生的台费 | 约150,000元 | 按人均贡献台费3000元估算 |
| 积分消耗去向 | | |
| - 兑换高毛利商品（成本30%） | 占80% → 4000分×30% = 1200元 | 用户兑换价值4000元商品，球房实际成本1200元 |
| - 实际提现 | 占20% → 1000分 = 1000元 | 少数玩家提现 |
| 球房总实际支出 | 2200元 | 商品成本 + 提现金额 |
| 球房净收益 | 约147,800元 | 台费收入150,000 - 支出2,200 = 147,800元 |
| 投资回报率 | 超67倍 | 每1元积分成本，带来超67元净收入 |

> 注：实际数据因球房运营水平而异，此模型为保守估算。

### 4.2 积分成本控制原则

- **积分产出比例可调：** 球房可根据淡旺季、台费价格灵活调整
- **提现限制：** 设置提现门槛，引导积分在店内循环
- **积分有效期：** 可设置积分有效期（如12个月），到期未使用则自动清零，刺激消耗

## 五、公平竞技与风控

### 5.1 系统监测

- 自动标记异常对局：如同一对手1小时内对战3次、固定比分模式
- 人工复核：对可疑对局，球房有权取消违规积分

### 5.2 诚信分体系

- 每位玩家初始100分诚信分
- 被确认刷分、消极比赛等违规行为，视情节扣5-50分
- 诚信分低于60分时，无法参与排行榜竞争及积分提现

### 5.3 举报通道

玩家可通过小程序或社群举报可疑行为，核实后举报人可获得50积分奖励。

## 六、多球房流通与未来展望

### 6.1 跨店流通（第二阶段）

当城市内有多个合作球房后，积分可升级为区域通用券：

- 用户在A球房获得的积分，可在B球房兑换商品/服务
- 提现仍需回原获得球房，防止套利
- 球房间通过平台进行积分结算

### 6.2 生态扩展（第三阶段）

当积分池足够大、流通足够稳定时：

- **平台信用：** 向优质球房提供"积分信用额度"，先发放后结算
- **数据服务：** 向器材商提供基于积分消耗的用户偏好数据报告
- **金融化：** 积分可用于购买线上课程、全国赛事报名费，成为台球垂直领域的"通用货币"

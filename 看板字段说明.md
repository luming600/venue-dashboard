# 天梯赛数据看板字段说明

## 一、CEO/用户运营看板字段

### 1. 核心指标卡片

| 字段名称 | 定义说明 | 计算方式 |
|---------|---------|---------|
| 总用户数 | 参与天梯赛的累计用户总数 | 去重统计所有注册用户 |
| 月活跃用户 | 近30天内有比赛记录的用户数 | 统计30天内有场次记录的去重用户 |
| 总场次 | 平台累计比赛场次总数 | 所有用户场次累加 |
| 平均胜率 | 全平台用户的平均胜率 | 所有用户胜率求平均 |
| 球馆数 | 接入天梯赛的球馆数量 | 去重统计球馆 |
| 最高连胜 | 平台最高连胜记录 | 取所有用户连胜最大值 |

### 2. 段位分布

| 段位名称 | 说明 |
|---------|------|
| 最强杆王 | 最高段位，代表顶尖玩家 |
| 校球钻石 | 高级段位，技术娴熟 |
| 加塞铂金 | 中高级段位，有一定技术水平 |
| 走位黄金 | 中级段位，基础扎实 |
| 翻袋白银 | 初级段位，正在成长 |
| 清台青铜 | 入门段位，新玩家居多 |

### 3. 用户分层（RFM模型）

| 分层名称 | 定义说明 | 判定标准 |
|---------|---------|---------|
| 核心用户 | 高价值、高活跃、高胜率用户 | 近7天活跃 + 场次≥50 + 胜率≥70% |
| 活跃用户 | 经常参与比赛的用户 | 近15天活跃 + 场次≥30 |
| 潜力用户 | 有成长空间的用户 | 近30天活跃 + 胜率≥60% + 场次<30 |
| 流失风险 | 曾经活跃但近期不活跃的用户 | 近30天无活动 + 历史场次≥30 |
| 普通用户 | 不符合以上条件的用户 | 默认分类 |

### 4. 地区分布

| 字段名称 | 说明 |
|---------|------|
| 省份用户分布 Top10 | 按用户数排序的省份排行 |
| 城市用户分布 Top15 | 按用户数排序的城市排行 |

### 5. 球馆分析

| 字段名称 | 说明 |
|---------|------|
| 球馆活跃度 Top10 | 按比赛场次排序的球馆排行 |

### 6. 趋势分析

| 字段名称 | 说明 |
|---------|------|
| 活跃用户（日） | 当日有比赛记录的用户数 |
| 新增用户（日） | 当日新注册的用户数 |
| 场次（日） | 当日比赛总场次 |
| 新增用户（周） | 该周新注册用户数 |
| 总场次（周） | 该周比赛总场次 |
| 活跃用户（周） | 该周有比赛记录的用户数 |

### 7. 用户表格字段

| 字段名称 | 说明 |
|---------|------|
| 排名 | 综合评分排名（积分×0.5 + 胜率×10 + 场次×2） |
| 用户ID | 用户唯一标识 |
| 昵称 | 用户显示名称 |
| 省份 | 用户所在省份 |
| 城市 | 用户所在城市 |
| 球馆 | 用户常去的球馆 |
| 段位 | 用户当前段位等级 |
| 积分 | 用户天梯积分 |
| 场次 | 用户累计比赛场次 |
| 胜率 | 用户比赛胜率（胜场/总场次×100%） |
| 连胜 | 用户当前或最高连胜场次 |
| 分层 | 用户所属分层类型 |

## 二、球房老板看板字段

### 1. 核心指标卡片

| 字段名称 | 定义说明 | 计算方式 |
|---------|---------|---------|
| 本店用户数 | 在本球馆打过球的累计用户数 | 筛选venue=本店的用户 |
| 本店月活跃 | 近30天在本店有比赛记录的用户 | 筛选本店用户中30天内有活动 |
| 本店总场次 | 本球馆累计比赛场次 | 本店所有用户场次累加 |
| 本店平均胜率 | 本店用户的平均胜率 | 本店用户胜率求平均 |
| 本店最高连胜 | 本店用户最高连胜记录 | 本店用户连胜最大值 |
| 区域排名 | 本球馆在区域内的活跃度排名 | 按场次/用户数在区域内排序 |

### 2. 周边球馆活跃度排名

| 字段名称 | 说明 |
|---------|------|
| 球馆名称 | 周边球馆名称（本店高亮显示） |
| 活跃度 | 球馆用户数或场次 |

## 三、通用字段说明

### 段位等级对照表

| 段位 | 积分范围 | 技术等级 |
|-----|---------|---------|
| 最强杆王 | 5000+ | 大师级 |
| 校球钻石 I-IV | 1500-5000 | 专家级 |
| 加塞铂金 I-IV | 600-1500 | 高手级 |
| 走位黄金 I-IV | 200-600 | 进阶级 |
| 翻袋白银 I-IV | 50-200 | 初级 |
| 清台青铜 I-IV | 0-50 | 入门级 |

### 用户分层运营建议

| 分层 | 运营策略 |
|-----|---------|
| 核心用户 | 重点维护，提供专属权益，防止流失 |
| 活跃用户 | 促进活跃，引导向核心用户转化 |
| 潜力用户 | 培养习惯，增加比赛频次 |
| 流失风险 | 召回唤醒，推送活动通知 |
| 普通用户 | 引导参与，提升活跃度 |
